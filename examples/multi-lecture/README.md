# Multi-Lecture Source Review Example

This example collection shows the expected source-review layout for collection mode.

```text
lectures/
  raw-course.txt
  01-introduction/
    raw-lecture.txt
    lecture.template.md
  02-core-concepts/
    raw-lecture.txt
    lecture.template.md
```

Per-lecture `raw-lecture.txt` files are the primary source evidence for each lecture. `lectures/raw-course.txt` is shared context when one course source was split into multiple lectures.
